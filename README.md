# Auto Pause YouTube Extension

This Chrome extension automatically pauses YouTube videos when you switch tabs (hide the tab) and resumes them when you switch back.

## Installation

1.  Open Chrome and navigate to `chrome://extensions`.
2.  Enable **Developer mode** (toggle in the top right corner).
3.  Click **Load unpacked**.
4.  Select the `auto_youtube_pauser` folder in this directory:
    `C:\Users\USER\.gemini\antigravity\scratch\auto_youtube_pauser`
5.  The extension is now installed and active on YouTube!

## Usage

-   Open a YouTube video.
-   Switch to another tab -> Video pauses.
-   Switch back to the YouTube tab -> Video resumes.
